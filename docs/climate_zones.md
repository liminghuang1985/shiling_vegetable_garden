# 时令菜园 - 中国气候区划分方案

> 来源：黄黎明确认版（2026-04-04）
> 说明：基于无霜期+积温+地理位置的综合分区，更适合种菜指导

---

## 最终确认的气候区划分

### 1区：东北寒冷区
**范围**：黑龙江、吉林、内蒙古东部（呼伦贝尔/兴安盟等）

| 参数 | 数值 |
|------|------|
| 无霜期 | <150天 |
| 积温 | <3400℃ |
| 代表城市 | 哈尔滨、长春、沈阳、呼和浩特、大庆、齐齐哈尔 |
| 种植特点 | 一年一熟；5-9月露地种菜；冬季极寒无法种菜 |

**适种季节**：5月-9月（盛夏季节）

---

### 2区：华北温带
**范围**：北京、天津、河北、山东、山西、河南（北部）、江苏（北部）、安徽（北部）

| 参数 | 数值 |
|------|------|
| 无霜期 | ~200天 |
| 积温 | 3400-5500℃ |
| 代表城市 | 北京、天津、济南、青岛、郑州、石家庄、太原、保定、唐山 |
| 种植特点 | 两年三熟或一年两熟；露地4-10月；设施农业发达 |

**适种季节**：3月-11月（部分地区需覆膜防寒）

---

### 3区：华中亚热带
**范围**：河南（南部）、江苏、安徽（南部）、湖北、湖南、江西、浙江、上海

| 参数 | 数值 |
|------|------|
| 无霜期 | 250-275天 |
| 积温 | 4500-6500℃ |
| 代表城市 | 武汉、长沙、南昌、合肥、南京、杭州、上海、苏州、成都（东部） |
| 种植特点 | 一年两熟；亚热带季风气候；梅雨季注意排水 |

**适种季节**：2月-12月（部分耐寒菜冬季可种）

---

### 4区：华南高温多雨区
**范围**：广东、广西、福建、海南、台湾、香港、澳门

| 参数 | 数值 |
|------|------|
| 无霜期 | 300天+（部分地区全年无霜） |
| 积温 | >6500℃ |
| 代表城市 | 广州、深圳、福州、厦门、南宁、海口、南昌（南部）、桂林、珠海 |
| 种植特点 | 热带/亚热带气候；全年可种；雨季注意病害防控 |

**适种季节**：全年（但需避开7-9月台风）

---

### 5区：西南高原区
**范围**：云南、贵州、四川（盆地以外）、西藏（东南部）、重庆

| 参数 | 数值 |
|------|------|
| 无霜期 | 差异大（昆明全年/拉萨<50天） |
| 积温 | 差异大（2000-6000℃） |
| 代表城市 | 昆明、贵阳、拉萨、丽江、大理、成都（盆地） |
| 种植特点 | "一山有四季"；海拔差异大；高原日照强但温度不高 |

**适种季节**：因海拔差异大，需按具体城市/海拔调整

---

### 6区：西北干旱区
**范围**：新疆、甘肃、宁夏、青海（东部）、陕西（北部）、内蒙古（西部）

| 参数 | 数值 |
|------|------|
| 无霜期 | 差异大（甘肃180天/新疆北部100-150天） |
| 积温 | 差异大 |
| 代表城市 | 乌鲁木齐、兰州、银川、西宁、西安（北部）、呼和浩特（西部） |
| 种植特点 | 干旱少雨；日照充足；需要灌溉；有暖棚可全年种菜 |

**适种季节**：甘肃/宁夏4-10月；新疆5-9月；需配合灌溉

## 与标准农业气候带的对应关系

| 本App气候区 | 对应标准气候带 | 主要区别 |
|------------|--------------|---------|
| 1区-东北寒冷区 | 寒温带+中温带 | 合并为一个区，强调无霜期短 |
| 2区-华北温带 | 暖温带+中温带 | 合并为一个区，考虑华北平原 |
| 3区-华中亚热带 | 亚热带（北/中） | 包含长江中下游 |
| 4区-华南高温多雨区 | 热带+亚热带（南） | 强调全年可种 |
| 5区-西南高原区 | 高原气候+亚热带 | 独特的高原区 |
| 6区-西北干旱区 | 暖温带+中温带 | 强调干旱特征 |

## 种植日历设计说明

每个气候区的种植日历以"月份"为横轴，以"蔬菜"为纵轴，标注该气候区该月份适合种植的蔬菜。

日历需区分：
- **春播**（2-4月）
- **夏播**（5-7月）
- **秋播**（8-10月）
- **冬播**（11-1月，仅部分南方区）
- **全年可种**（华南4区）

阳台种菜在日历基础上叠加"阳台朝向"筛选。
