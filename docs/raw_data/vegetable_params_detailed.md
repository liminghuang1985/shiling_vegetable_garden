# 四大主菜详细种植参数
# 来源：多个农业网站交叉验证
# 整理时间：2026-04-04

---

## 1. 番茄（Tomato）

| 参数 | 数值 |
|------|------|
| **发芽温度** | 20-28℃，低于15℃或高于30℃发芽率急剧下降 |
| **生长温度** | 15-30℃，最佳20-25℃；低于10℃停止生长，低于5℃可能受害，-2至-1℃可能冻死；高于30℃同化降低，高于35℃生殖生长受干扰 |
| **成熟天数** | 110-170天（整个生长周期） |
| **pH值** | 6.0-6.5 |
| **株距** | 大果型30-40cm；小果型/樱桃番茄25-30cm |
| **行距** | 大果型70-80cm；小果型60-70cm |
| **播种深度** | 约1cm |
| **光照** | 喜阳（fullSun），需充足日照 |
| **土壤** | 疏松肥沃，排水良好 |
| **肥料** | 基肥：腐熟有机肥；追肥：结果期增施磷钾肥 |
| **注意事项** | 不耐连作；需整枝打杈；注意防治晚疫病 |

## 2. 黄瓜（Cucumber）

| 参数 | 数值 |
|------|------|
| **发芽温度** | 最低13℃，最适30℃；高于35℃发芽率降低 |
| **生长温度** | 25-32℃（最佳）；0-2℃冻死，5-10℃可能受害；最佳地温20-23℃ |
| **成熟天数** | 50-70天（从种子到结果） |
| **pH值** | 5.5-7.2，最适6.5 |
| **株距** | 常规25-35cm；大棚30-45cm；搭架25-30cm |
| **行距** | 露地常规60-70cm；大棚单行120-140cm；双行大行80-100cm，小行40-50cm |
| **播种深度** | 约1.5-2cm |
| **光照** | 喜阳（fullSun） |
| **土壤** | 疏松肥沃，保水性好但排水良好 |
| **肥料** | 基肥：有机肥为主；追肥：薄肥勤施，结瓜期增施钾肥 |
| **注意事项** | 喜湿润但不耐涝；需搭架或吊蔓；注意防治霜霉病、白粉病 |

## 3. 茄子（Eggplant）

| 参数 | 数值 |
|------|------|
| **发芽温度** | 20-25℃（部分资料25-30℃），最适30℃ |
| **发芽天数** | 5-14天 |
| **生长温度** | 生育适温20-30℃；开花结果适温25-30℃；幼苗期白天25-30℃，夜间15-20℃ |
| **成熟天数** | 开花到商品成熟13-15天；开花到最终成熟50-60天；早熟品种开花后20-25天可采收 |
| **pH值** | 6.8-7.3 |
| **株距** | 50-80cm |
| **行距** | 70cm |
| **播种深度** | 约1cm |
| **光照** | 喜阳（fullSun） |
| **土壤** | 疏松肥沃，排水良好 |
| **肥料** | 基肥：有机肥+磷钾肥；追肥：结果期重施磷钾肥 |
| **注意事项** | 不耐涝；需整枝；注意防治黄萎病、绵疫病 |

## 4. 辣椒（Pepper）

| 参数 | 数值 |
|------|------|
| **发芽温度** | 25-30℃（最佳25-28℃） |
| **发芽天数** | 5-7天 |
| **生长温度** | 白天25-30℃，夜晚15-18℃；整体适温20-28℃ |
| **成熟天数** | 60-90天（播种到成熟）；果实商品成熟（绿熟期）25-40天（早熟25-30天，晚熟35-40天）；生理成熟需45-60天 |
| **pH值** | 5.5-6.8，最适约6.5 |
| **株距** | 青椒25-30cm；早熟品种26-33cm；晚熟品种50-60cm |
| **行距** | 青椒50-60cm；早熟品种40-50cm；晚熟品种66-73cm |
| **播种深度** | 约1cm |
| **光照** | 喜阳（fullSun） |
| **土壤** | 疏松肥沃，排水良好 |
| **肥料** | 基肥：有机肥；追肥：结果期增施磷钾，少施氮肥 |
| **注意事项** | 不耐旱也不耐涝；注意防治炭疽病、病毒病 |
