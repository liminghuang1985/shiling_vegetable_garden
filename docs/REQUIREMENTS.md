## 一、产品定位

**一句话描述**：基于中国气候带的时令蔬菜种植指南，帮助城市人找到"现在能种什么菜"。

**核心场景**：春播秋播 —— 用户打开 App，立刻知道这个月在家适合种什么。

**目标用户**：全人群都市种菜爱好者
- 阳台种菜族（20-45岁，上班族，种植经验少）
- 退休种菜族（50岁+，有闲，种植经验丰富）

## 三、用户路径

### 首次打开
```
欢迎页 → 选择城市 + 阳台朝向 → 识别气候带 → 首页
```

### 正常流程
```
首页
  ├─ 当前气候带 + 月份 + 朝向
  ├─ 「现在可以种」（分类展示蔬菜卡片）
  ├─ 搜索框
  └─ 「我的菜园」入口

点击蔬菜卡片 → 详情页
  ├─ 基本信息（分类/温度/光照）
  ├─ 种植要点（土壤/肥料/播种/间距）
  ├─ 周期（发芽→成熟天数）
  ├─ 营养价值
  └─ 注意事项

我的菜园
  ├─ 添加蔬菜（从时令列表选或手动搜索）
  ├─ 生长记录（播种日期 → 预计成熟）
  ├─ 提醒管理（浇水/施肥/采收）
  └─ 采收完成 → 标记结束
```

## 四、数据规格

### 4.1 蔬菜数量
- MVP：60种常见蔬菜
- 来源：预置 JSON 数据包

### 4.2 气候带划分（中国标准）

| 气候带 | 代表城市 | 积温范围 |
|--------|---------|---------|
| 寒温带 | 哈尔滨、长春、沈阳、呼和浩特 | <3400℃ |
| 中温带 | 北京、天津、石家庄、济南 | 3400-4500℃ |
| 暖温带 | 郑州、西安、兰州、青岛 | 4500-5500℃ |
| 亚热带 | 武汉、长沙、南昌、合肥、南京 | 5500-8000℃ |
| 热带 | 广州、深圳、福州、南宁、海口 | >8000℃ |
| 高原气候 | 昆明、拉萨、西宁 | 特殊 |

### 4.3 阳台朝向

| 朝向 | 光照特点 | 适合蔬菜 |
|------|---------|---------|
| 南向 | 全日照，阳光充足 | 几乎所有蔬菜（瓜果类最佳） |
| 东向 | 半日照，上午光照 | 喜光耐阴蔬菜（油菜、韭菜） |
| 西向 | 半日照，下午光照 | 耐高温爬藤类（丝瓜、苦瓜） |
| 北向 | 光照不足 | 耐阴蔬菜（莴苣、芦笋、香椿） |

### 4.4 云端更新频率
- 季度更新（每3个月一次）
- 数据内容：蔬菜详情更新 + 种植日历调整 + 新增城市
